Android Application
===================

How to use the Android App
--------------------------

-   **To use any feature of the PSLab Android application first follow
    these 2 steps**:
    1.  Connect PSLab board with your Android smartphone using OTG cable
    2.  Open [PSLab Android application](https://play.google.com/store/apps/details?id=io.pslab)
